WarehouseManager API: Link Relations

Standard link relations

  • self: carried by all resources returned by the API. Points to the resource itself. It is also carried by all Items in Collections, pointing at the related resource.
  • profile: points to the related proflie of the current resource.
  • collection: carried by Item resources, points to the related Collection.
  • edit: carried by Item resources, points to itself, specifying PUT method and the appropriate JSON schema to use for the request

Custom link relations

  • warehouse: points to the Warehouse resource that contains the current resource.
  • items: points to the Items Collection resource that contains the current resource.
  • item: points to the Item resource that contains the current resource.
  • product: points to the Product resource that contains the current resource.
  • products: points to the Products Collection resource that contains the current resource.
